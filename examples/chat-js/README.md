## Chat example in JS

### Requirements
- Node 18
- An OPENAI_API_KEY
    - Set it in the `.env` file for the server and this project (where the `.env.example` are)

### How to run

In the server root, run:
```bash
docker-compose build
docker-compose up
```

In this folder run:
```bash
npm i
npm start
```

Start chatting!
